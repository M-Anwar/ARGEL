ARGEL MARKETING

ARGEL takes advantage of the current rise in crowd analytics to create a convenient but effective platform for targeting ads towards the audience present in a given public space. ARGEL is a distribution infrastructure for advertisements in physical media using machine learning and data mining techniques. The system displays ads targeted towards the crowd in real time by extracting visual features and applying learning algorithms to the data.
